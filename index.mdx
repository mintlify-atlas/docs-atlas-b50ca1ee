OmniVoice is a state-of-the-art zero-shot multilingual text-to-speech model built on a diffusion language model architecture. It supports over 600 languages, delivers real-time-factor as low as 0.025, and lets you clone voices or design entirely new ones without any fine-tuning.

## Get started

<Steps>
  <Step title="Install OmniVoice">
    Install via pip with PyTorch for your platform:

    ```bash
    pip install torch==2.8.0+cu128 torchaudio==2.8.0+cu128 --extra-index-url https://download.pytorch.org/whl/cu128
    pip install omnivoice
    ```
  </Step>
  <Step title="Load the model">
    Download the pretrained model from Hugging Face:

    ```python
    from omnivoice import OmniVoice
    import torch

    model = OmniVoice.from_pretrained(
        "k2-fsa/OmniVoice",
        device_map="cuda:0",
        dtype=torch.float16
    )
    ```
  </Step>
  <Step title="Generate speech">
    Clone a voice or describe one with text attributes:

    ```python
    import torchaudio

    # Voice cloning
    audio = model.generate(
        text="Hello, this is a test of zero-shot voice cloning.",
        ref_audio="ref.wav",
        ref_text="Transcription of the reference audio.",
    )
    torchaudio.save("out.wav", audio[0], 24000)
    ```
  </Step>
  <Step title="Try the web demo">
    Launch the interactive Gradio UI locally:

    ```bash
    omnivoice-demo --ip 0.0.0.0 --port 8001
    ```

    Or try it on [Hugging Face Spaces](https://huggingface.co/spaces/k2-fsa/OmniVoice) without any installation.
  </Step>
</Steps>
